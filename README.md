# CadUsu

<p>Aplicação web para cadastro de usuários desenvolvido para o desafio técnico da Softplan</p>


* Para buildar e rodar os testes da API:
<pre>cd cadusu</pre>
<pre>mvn clean install</pre>

## Instruções para execução:

* Instale o docker engine e docker compose 

* Execute o seguinte comando no diretório raiz deste projeto:
<pre>docker-compose up -d<pre>

* Os endereços para acesso às telas (frontend) e da API (backend) serão os seguintes:

<p>
API: http://localhost:8081/swagger-ui<br/>
Front: http://localhost
</p>
